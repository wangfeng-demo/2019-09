### AJAX的状态码
> xhr.readyState 获取状态码
- UNSEND 0 : 未发送（创建一个XHR，初始状态是0）
- OPENED 1 ：已经打开（执行了xhr.open）
- HEADERS_RECEIVED 2 : 响应头信息已经返回给客户端（发送请求后，服务器会依次返回响应头和响应主体的信息）
- LOADING 3 : 等待服务器返回响应内容
- DONE 4 : 响应主体信息已经返回给客户端
