### less
> http://lesscss.cn/
> less是css预编译语言，和其类似的预编译器还有：sass、stylus等；而所谓的预编译，其实是把css这种标记语言，按照面向对象（编程语言）的方式进行编写（有变量、函数、判断等操作），但是这种写法浏览器不能直接的识别，需要我们把其再编译为正常的css代码才可以；
